# Summit Adventures - Adventure Park Website

A modern, responsive landing page for Summit Adventures - India's largest adventure park.

## Features

- **Dark Theme** with striking yellow/gold accents
- **Fixed Navigation Header** with smooth scroll
- **Full-screen Hero Section** with stats counter
- **Activity Cards** with difficulty badges
- **Pricing Cards** with featured/popular highlights
- **Testimonials Section** with star ratings
- **Responsive Design** for all devices
- **Smooth Animations** on scroll

## File Structure

```
Frontend/
├── index.html              # Main HTML file
├── css/
│   ├── variables.css       # CSS custom properties
│   ├── style.css           # Main styles
│   └── responsive.css      # Mobile/tablet styles
├── js/
│   ├── main.js             # Core functionality
│   └── animations.js       # Scroll animations
├── images/
│   ├── hero-bg.jpg         # Hero background
│   └── activities/         # Activity images
│       ├── flying-fox.jpg
│       ├── bungee-jump.jpg
│       └── buggy-ride.jpg
└── README.md
```

## Images Required

Add the following images to the `images/` folder:

1. **Hero Background** (`images/hero-bg.jpg`)
   - Recommended size: 1920x1080px
   - Adventure/outdoor themed image with warm tones

2. **Activity Images** (in `images/activities/`)
   - `flying-fox.jpg` - Zipline/flying fox activity
   - `bungee-jump.jpg` - Bungee jumping activity
   - `buggy-ride.jpg` - Off-road buggy activity
   - Recommended size: 800x600px each

## Setup

1. Clone or download the project
2. Add your images to the `images/` folder
3. Open `index.html` in a browser
4. For development, use a local server (e.g., Live Server extension in VS Code)

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Customization

### Colors
Edit `css/variables.css` to change the color scheme:
- `--color-primary`: Main accent color (default: #F5A623)
- `--color-bg-dark`: Dark background (default: #0D0D0D)

### Typography
The site uses:
- **Bebas Neue** for headings
- **Inter** for body text
